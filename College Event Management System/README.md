# College Event Management System

## Project Overview

This project is a responsive college event management website that allows students to explore upcoming campus events, register for events, review event schedules, browse a photo gallery, and get answers from frequently asked questions. The site is built using Bootstrap components for a mobile-friendly user experience and includes a registration workflow with success feedback.

## Pages Included

- **Home** — Banner, highlights, featured event calls to action, and a Bootstrap carousel.
- **Events** — Six event cards with images, dates, venues, and register buttons.
- **Registration** — Student registration form with event selection and success alert.
- **Event Schedule** — Responsive striped and hoverable event schedule table.
- **Gallery** — Event photographs displayed in responsive cards and grid layout.
- **FAQ** — Accordion-based frequently asked questions section.
- **Contact** — Contact form, coordinator details, and social media links.
