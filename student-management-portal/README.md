# Student Management Portal

This folder contains a static student management portal website built with HTML, CSS, and JavaScript.

## Structure

- `index.html` - Main landing page for the portal.
- `about.html` - About page describing the portal and its purpose.
- `contact.html` - Contact page with a message form and office details.
- `dashboard.html` - Dashboard page for student or admin functions.
- `registration.html` - Student registration page.
- `css/` - Stylesheet folder containing the site styling.
- `js/` - JavaScript folder for interactive behavior.
- `images/` - Image assets used across the portal.
- `screenshots/` - Screenshots showing page previews.

## Usage

1. Open any `.html` file in a web browser.
2. Use the navigation menu to move between pages.
3. Edit the HTML, CSS, or JavaScript files to customize the portal.

## Notes

- The portal is static and does not require a backend server.
- For local changes, refresh the browser after editing files.
