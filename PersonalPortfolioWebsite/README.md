# Personal Portfolio Website

This is a simple professional personal portfolio website scaffold. It includes:

- `index.html` — single page with sections: Home, About, Skills, Projects, Achievements, Contact
- `css/style.css` — minimal custom styles
- `js/script.js` — simple contact form handling
- `assets/resume.pdf` — placeholder resume (replace with your real resume)
- `img/` — placeholder images

Open `index.html` in a browser to view the site.
